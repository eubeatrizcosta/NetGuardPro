# 📖 Sobre o projeto

O **NetGuard Pro** é uma plataforma desenvolvida para monitoramento e gerenciamento de infraestrutura de redes, permitindo que equipes de TI acompanhem a saúde dos servidores, dispositivos e serviços em tempo real.

O sistema foi pensado para reduzir o tempo de resposta a incidentes, melhorar a disponibilidade dos serviços e facilitar a tomada de decisões através de dashboards intuitivos e alertas automáticos.

# ⚙️ Instalação

Clone o projeto

```bash
git clone https://github.com/seuusuario/netguard-pro.git
```

Entre na pasta

```bash
cd netguard-pro
```

Crie um ambiente virtual

```bash
python -m venv venv
```

Instale as dependências

```bash
pip install -r requirements.txt
```

Execute

```bash
python app.py
```

# 💡 Caso de Uso

Imagine uma empresa com diversos servidores espalhados por diferentes filiais.

O NetGuard Pro permite:

- 📊 acompanhar o consumo de CPU;
- 💾 monitorar memória e armazenamento;
- 🌐 identificar problemas de rede;
- 🔔 receber alertas automáticos;
- 📈 gerar relatórios de disponibilidade.

Isso reduz o tempo de resposta a incidentes e melhora a estabilidade da infraestrutura.
